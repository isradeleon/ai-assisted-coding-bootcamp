# Vibe Task Manager (Electron)

Modern Electron desktop app for creating and managing tasks with these statuses:

- `Pending`
- `In progress`
- `Done`

## Run

From `task-manager-app/`:

```bash
npm install
npm start
```

## Features

- Create tasks
- Move tasks between statuses (`Pending` → `In progress` → `Done`)
- Delete tasks
- Local persistence (stored on your machine via `electron-store`)
