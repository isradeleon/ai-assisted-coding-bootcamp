# v0-portfolio-website

This is a simple portfolio website built with HTML, CSS, and JavaScript.

## Folder Structure

The folder structure has been updated to be more organized:

- `index.html`: The main HTML file.
- `css/`: This directory contains all the CSS files, separated by component.
  - `main.css`: Global styles, variables, and theme settings.
  - `navigation.css`: Styles for the navigation bar.
  - `hero.css`: Styles for the hero section.
  - `about.css`: Styles for the about section.
  - `experience.css`: Styles for the experience section.
  - `projects.css`: Styles for the projects section.
  - `tech-stack.css`: Styles for the tech stack section.
  - `contact.css`: Styles for the contact section.
  - `footer.css`: Styles for the footer.
  - `responsive.css`: Styles for mobile devices.
  - `animations.css`: Styles for scroll animations.
- `js/`: This directory contains all the JavaScript files.
  - `main.js`: Contains all the JavaScript code for the website.

## How to use

Simply open the `index.html` file in your browser to view the website.
